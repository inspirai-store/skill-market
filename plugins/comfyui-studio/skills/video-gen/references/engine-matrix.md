## FramePack 详细参数

### 基础信息

| 项目 | 值 |
|------|-----|
| 开发者 | lllyasviel |
| 最低显存 | 6GB |
| 推荐显存 | 8-12GB |
| 支持模式 | 图生视频 |
| 最大时长 | 60秒+（分段拼接可更长） |
| 原理 | 逐帧扩散，帧打包压缩显存 |

### 参数推荐

| 参数 | 低显存（6-8GB） | 中显存（8-12GB） | 高显存（12GB+） |
|------|---------------|-----------------|----------------|
| 分辨率 | 512x512 | 512x768 | 768x768 |
| 总帧数 | 48（2秒@24fps） | 120（5秒@24fps） | 360（15秒@24fps） |
| 步数 | 15-20 | 20-25 | 25-30 |
| CFG / guidance | 7.0 | 7.0-9.0 | 7.0-9.0 |
| motion_strength | 0.3-0.7 | 0.3-1.0 | 0.3-1.0 |
| 输出帧率 | 24fps | 24fps | 24fps |

### 优缺点

**优点：**
- 极低显存要求（6GB 可运行）
- 支持超长视频（60秒+）
- 视频质量高，运动自然
- 支持首帧图片引导

**缺点：**
- 生成速度慢（逐帧生成）
- 目前仅支持图生视频
- 高分辨率需要更多显存
- 较新，生态仍在完善

## AnimateDiff V3 详细参数

### 基础信息

| 项目 | 值 |
|------|-----|
| 开发者 | guoyww |
| 最低显存 | 8GB |
| 推荐显存 | 8-12GB |
| 支持模式 | 文生视频、图生视频 |
| 最大时长 | 2-4秒（16-32帧@8fps） |
| 原理 | 在 SD 基础上注入时序运动模块 |

### 参数推荐

| 参数 | SD 1.5 模式 | SDXL 模式（实验性） |
|------|-----------|-------------------|
| 分辨率 | 512x512 / 512x768 | 768x768 / 768x1024 |
| 帧数 | 16（推荐）/ 24 / 32 | 16 |
| 上下文长度 | 16 | 16 |
| 上下文重叠 | 4 | 4 |
| 步数 | 20-25 | 25-30 |
| CFG | 7.0-8.0 | 6.0-8.0 |
| 采样器 | dpmpp_2m + karras | dpmpp_2m + karras |
| motion_scale | 1.0（默认） | 0.8-1.0 |
| 输出帧率 | 8fps（原始）→ 24fps（插值后） | 8fps → 24fps |

### 运动模块选择

| 模块 | 适用模型 | 特点 |
|------|---------|------|
| mm_sd15_v3 | SD 1.5 | 最新版，运动质量最高 |
| mm_sd15_v2 | SD 1.5 | 稳定版，兼容性好 |
| mm_sdxl_v10_beta | SDXL | 实验性，效果不稳定 |

### 优缺点

**优点：**
- 生成速度快
- 生态成熟，教程丰富
- 支持 ControlNet 精确控制
- 支持 LoRA 保持风格
- 可做循环动画

**缺点：**
- 时长严重受限（2-4秒）
- 原始帧率低（8fps），需后期插值
- 大运动场景容易变形
- 基于 SD 1.5，分辨率受限

## 引擎选择速查表

### 按显存选择

| 可用显存 | 首选引擎 | 备选引擎 |
|---------|---------|---------|
| 6-7GB | FramePack | — |
| 8GB | FramePack | AnimateDiff V3 / LTX Video |
| 10-12GB | Wan 2.1 量化 | FramePack |
| 12-16GB | Wan 2.1 FP8 | FramePack（长视频） |
| 16-24GB | Wan 2.1 FP16 | Wan 2.2（紧凑） |
| 24GB+ | Wan 2.2 14B | Wan 2.1 FP16 |

### 按需求选择

| 需求 | 推荐引擎 | 说明 |
|------|---------|------|
| 长视频（>10秒） | FramePack | 唯一支持超长视频的引擎 |
| 快速预览 | AnimateDiff / LTX | 几十秒即可出结果 |
| 最高质量 | Wan 2.2 14B | 电影级效果 |
| 循环动画 | AnimateDiff | 天生支持循环 |
| 中文提示词 | Wan 系列 | 对中文理解最好 |
| 精确动作控制 | AnimateDiff + ControlNet | ControlNet 生态最成熟 |

### 按输入类型选择

| 输入 | 推荐引擎 | 说明 |
|------|---------|------|
| 单张图片 → 视频 | FramePack / Wan i2v | 图生视频 |
| 文字描述 → 视频 | Wan t2v / AnimateDiff | 文生视频 |
| 首尾帧 → 视频 | Wan i2v | 插值生成 |
| 视频 → 风格转换 | AnimateDiff + ControlNet | 逐帧重绘 |
