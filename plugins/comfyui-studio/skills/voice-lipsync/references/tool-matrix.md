## 硬件配置推荐

### 8GB 显存（如 RTX 4060/3060）

**推荐方案：** ChatTTS + Wav2Lip + GFPGAN
- TTS 和唇形同步分开运行（不同时占用显存）
- Wav2Lip 处理后用 GFPGAN 修复面部
- 适合 512px 分辨率输出

### 12GB 显存（如 RTX 4070/3060 12G）

**推荐方案：** F5-TTS + LatentSync
- 可以获得高质量输出
- TTS 和唇形同步需分开运行
- 适合 768-1024px 分辨率输出

### 16GB+ 显存（如 RTX 4080/4090）

**推荐方案：** GPT-SoVITS + LatentSync
- 完整高质量流水线
- 可同时加载多个模型
- 适合 1024px 分辨率输出
- 支持批量处理
