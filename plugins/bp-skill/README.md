# BP Skill - 最佳实践管理

记录验证通过的解决方案，跨项目复用，避免重复踩坑。

## 功能

- `/bp capture` - 记录新的最佳实践
- `/bp search <keyword>` - 搜索相关实践
- `/bp list [category]` - 列出所有或按分类列出
- `/bp apply <id>` - 应用实践到当前项目
- `/bp update <id>` - 更新已有实践
- `/bp delete <id>` - 删除过时实践

## 存储位置

`~/.inspirai/best-practices/`

## 安装

```bash
claude /install /path/to/bp-skill
```
